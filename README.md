# React Todo App

It is a simple and responsive Todo application built using React. This project showcases state management, and dynamic UI updates.

---

##  Features

-  Add new todos
-  Delete existing todos
-  Loading indicator while adding tasks
-  Empty state handling ("No Todos Available")
-  Clean UI using Bootstrap

---

## What I Learned

- React fundamentals (components, props, useState)
- Event handling in React
- Conditional rendering
- Managing and updating state dynamically

---

## Tech Stack

- React
- JavaScript
- Bootstrap
